About This Application

This application is designed to manage albums and their assigned photos. Users can view a list of albums, review details for each album, and view photos in each album.

Features

  • View a list of albums
  • View details of each album
  • Edit album titles
  • Delete albums
  • View photos in each album

Technologies Used

This application is built using Angular 17, TypeScript, and CSS for styling.
